# Banking Automation Agent

AI-powered system for processing banking emails and documents.

## Architecture
- Email processing → Document extraction → Classification → Response generation

## Key Constraints
- Use OPENAI api
- Keep UI simple with Streamlit
- Mock data only (no real email servers)
- SQLite for simplicity

## Code Style
- Type hints required
- Comprehensive error handling
- Logging for debugging
- Test coverage >80%
